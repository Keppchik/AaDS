## Запуск проекта
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/keppchik/AaDS.git
   ```
2. Перейдите в папку с проектом:
   ```bash
   cd AaDS/lab2
   ```
3. **Запуску всех задач**
   ```bash
    for script in */src/*.py; do PYTHONPATH=$(pwd) python "$script"; done
   ```
