## Запуск проекта
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/keppchik/AaDS.git
   ```
2. Перейдите в папку с проектом:
   ```bash
   cd AaDS
   ```
3. **Запуск всех задач**

   ```bash
   for script in lab7/*/src/*.py; do PYTHONPATH=$(pwd) python "$script"; done
   ```

## Тестирование
